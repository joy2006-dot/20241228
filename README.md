這是一個雙人對戰遊戲，以下是完整的操作說明：
玩家1 (綠色血條)
移動控制：
A - 向左移動
D - 向右移動
W - 跳躍
攻擊：
F - 發射子彈
子彈會朝著角色面向的方向發射
玩家2 (紅色血條)
移動控制：
← (左方向鍵) - 向左移動
→ (右方向鍵) - 向右移動
↑ (上方向鍵) - 跳躍
攻擊：
空白鍵 - 發射子彈
子彈會朝著角色面向的方向發射
遊戲規則
每個玩家初始血量為 100
被子彈擊中會減少 10 點血量
當任一玩家血量歸零時遊戲結束
按 ENTER 鍵可以重新開始遊戲
遊戲介面
左上角顯示玩家1的血條（綠色）
右上角顯示玩家2的血條（紅色）
血條下方顯示各玩家的戰績（勝場、敗場、平手）
背景有半透明的 'TKUET' 文字
祝您遊戲愉快！
